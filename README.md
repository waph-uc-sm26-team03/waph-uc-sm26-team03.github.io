# Overview

This project will develop a secure MiniFacebook web application using PHP, MySQL, JavaScript, HTML, and CSS. The application will allow users to register, authenticate, create posts, comment on posts, and communicate through a real-time chat system while following secure web programming principles.

---

# System Analysis

## High-level Requirements

- User registration
- User login/logout
- User profile management
- Password update
- Create posts
- Edit and delete own posts
- View and comment on posts
- Real-time chat between authenticated users
- Superuser account management
- Secure session management
- Protection against common web attacks

# Security Analysis

The application will follow secure programming practices including:

- Password hashing
- Secure session management
- Input validation
- Output encoding
- SQL Injection prevention using prepared statements
- Cross-Site Scripting (XSS) prevention
- Cross-Site Request Forgery (CSRF) protection
- Role-based authorization for superusers and regular users
